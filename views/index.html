Fun node projects for freecodecamp.org:

1. Micro-Timestamp:

the url route is at /getdate/:date where date is a number or "YYYY-MM-DD" provided format

2. Request Header Parser:

the url route is at /computerinfo, provides request header info

3. URL Shortener:

the url route is at /new_url/:url, creates a new url to that page on this website.

4. Image Search Abstraction Layer

the url route is at /getimage/:imageterm, which looks up an image. Note, I did not implement caching.

Repeated searches on the same term might fail...

the url route /getsearches/ produces the last 10 searches

5. File Metadata Microservice

the url route is at /file_size. Upload a file to get it's size.
